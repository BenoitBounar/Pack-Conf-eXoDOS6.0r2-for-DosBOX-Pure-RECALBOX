# Pack-Conf-eXoDOS6.0r2-for-DosBOX-Pure-RECALBOX

:fr: Pack de fichiers de configuration pour lancer automatiquement les jeux DOS issus du set eXoDOS 6.0r2 sur DOSBox Pure.
